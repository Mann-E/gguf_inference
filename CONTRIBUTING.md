# Contribution Guide

First, [fork this repo](https://github.com/Mann-E/gguf_inference/fork) to your own github account. Second, clone the fork on your own machine. 

If you intend to make a change, using _TO DO List_ and _Known Issues_ section of [README.md](README.md) are appreciated, and we're also open to new ideas as well. 

## How to make the change 

Make a new branch for your change:

```
git checkout -b my-awesome-change
```

And after making the change and pushing it, make a pull request to this repository. We'll review and if it's accepted, we'll merge that.
